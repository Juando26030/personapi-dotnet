# Archivos de librerías de cliente ignorados

Este proyecto usa librerías de cliente (jQuery, Bootstrap, etc.) que normalmente se restauran mediante:

```bash
dotnet restore
libman restore
```

O mediante Visual Studio que las descarga automáticamente.

Las librerías incluyen:
- Bootstrap 5.x
- jQuery 3.x
- jQuery Validation
- jQuery Validation Unobtrusive

Estas librerías NO se incluyen en el repositorio Git por su tamaño.
